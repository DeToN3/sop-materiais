## Compilação

```
gcc meu_arquivo_fonte.c -o nome_executavel
```

## Execução

```
./nome_executavel
```

## Execução com passagem de parâmetros

Cada parâmetro informado é separado com um espaço

```
./nome_executavel parametro_1 parametro_2
```
